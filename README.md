# MeetingTune

**MeetingTune** is a desktop app for managing and tuning your meeting experience.

## Download

Download the latest installer from the [Releases](../../releases) page.

| Version | Platform | Download |
|---------|----------|----------|
| 1.0.0   | Windows  | [MeetingTune Setup 1.0.0.exe](../../releases/latest) |

## Installation

1. Download `MeetingTune Setup 1.0.0.exe` from the [latest release](../../releases/latest).
2. Run the installer and follow the on-screen instructions.
3. Launch **MeetingTune** from your desktop or Start Menu.

## Requirements

- Windows 10 or later
